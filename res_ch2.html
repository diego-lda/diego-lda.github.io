Bitcoin Community Structures

This is the main chapter of my thesis. Using the transactions between addresses, I apply network econometrics to analyse the community structures of Bitcoin during the last 3 years. An initial contribution to the literature is the refinement of network measures in this context. A further contribution is the estimation of strategic interaction at the community level.

Applying existing methods, I take the raw transaction data and build a graph at the user level, aggregating addresses that belong to the same agent. Moreover, using a modularity optimising algorithm, communities of users are obtained. I combine this with known address data, which allows anonymous agents to be assigned a user type. Network statistics computed from these modified graphs contribute to the literature on community structure using blockchain data as well as its dynamics.

Beyond this, the weekly cyclicality phenomenon present in network usage is discussed. Heterogeneity at the community level is established using the Kolmogorov-Smirnov test on degree distributions from communities of differing types. This shows exchange-based communities are arranged differently than mining-based communities.
